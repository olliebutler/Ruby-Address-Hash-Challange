# Ruby-Address-Hash-Challange
A ruby method that takes a address hash and returns a string of the formatted address

Sample input 

{
  "subpremise" => "Apt 6",
  "house_number" => "22",
  "house_name" => nil,
  "street_line_1" => "Bailey Rd",
  "street_line_2" => nil,
  "town_or_city" => "Sale",
  "region" => "Greater Manchester",
  "postcode" => "M33 1AX"
}

Sample output

Apt 6, 
22, Bailey Rd,
Sale,
Greater Manchester,
M33 1AX
